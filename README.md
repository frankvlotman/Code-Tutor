# Code-Tutor

- **Lets users type and run Python code** directly inside the app.

- **Displays output** in a built-in results window, instead of the terminal.

- Uses **colour-highlighted syntax** (keywords like `def`, `print`, etc.).

- Supports **Tab indentation** to help kids learn proper Python structure.

- Provides a **safe, simplified environment** ideal for young learners.

- Encourages learning through **experimentation and instant feedback**.

- Helps children build confidence with **basic Python concepts** such as variables, printing, and simple functions.
